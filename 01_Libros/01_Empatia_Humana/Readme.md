# Arquitectura del Destino · Oráculo y Libro

## Constelación Campo Trinity — Libro VII

### Descripción

Este repositorio contiene dos experiencias complementarias:

1. **El libro (`index.html`)** : 112 aforismos organizados en 25 ejes temáticos. Puede leerse de forma lineal o por secciones. Cada aforismo incluye una pregunta para la reflexión personal o terapéutica.

2. **El oráculo (`oraculo.html`)** : una experiencia interactiva de 3 umbrales (intención, pausa, revelación). Cada consulta devuelve un aforismo al azar con su geometría animada asociada y un sonido contemplativo. El campo escucha, pero no guarda nada.

### Cómo usar

- **Modo libro:** abrí `index.html` en cualquier navegador. Explorá las secciones desplegables. Leé los aforismos en el orden que quieras.
- **Modo oráculo:** abrí `oraculo.html`. Escribí una palabra, un nombre o una sensación (no se guarda). Esperá 11 segundos. Recibí un aforismo y su pregunta. Podés repetir el ciclo las veces que necesites.

### Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Canvas animado (fondo dinámico)
- Web Audio API (tonos sutiles, requiere interacción del usuario para activarse)
- Fuentes: Cormorant Garamond, Cinzel, Source Sans 3

### Compatibilidad

- **Safari:** el oráculo puede presentar limitaciones con el audio y algunas geometrías debido a sus políticas de seguridad. Se recomienda Chrome, Firefox o Edge para la experiencia completa. El modo libro funciona sin problemas.
